# RickAndMorty (iOS)

This app uses [The Rick and Morty API](https://rickandmortyapi.com)

## Used

- MVVM
- UIKit
- SwiftUI

## Installation

1. Clone the repository:

```
git clone https://github.com/ighiba/rick-and-morty.git
```

2. Open the .xcodeproj file in Xcode.

3. Build and run the app

## Requirements

- iOS 15 or later.
- Xcode 14 or later.
